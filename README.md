# Tailwind CSS Practice Exercises

A comprehensive collection of 20 Tailwind CSS practice exercises demonstrating modern utility-first CSS techniques.

## 📚 Exercises

1. **Typography & Spacing** - Font sizes, weights, and spacing utilities
2. **Colorful Alert Boxes** - Success, warning, and danger alerts
3. **Simple Card with Shadow** - Card component with hover effects
4. **Circular Profile Badge** - User profile with avatar
5. **Interactive Buttons** - Primary, secondary, and outline buttons
6. **Responsive Two-Column Layout** - Mobile-first grid layout
7. **Product Grid (3-Columns)** - Responsive product showcase
8. **Hero Section with Overlay** - Full-width hero with text overlay
9. **Pricing Table Comparison** - Three-tier pricing cards
10. **Simple Navigation Bar** - Sticky navigation with blur effect
11. **Custom Flexbox Alignment** - Various flexbox patterns
12. **Feature List with Icons** - Icon-based feature cards
13. **Sticky Sidebar Layout** - Dashboard-style sidebar
14. **Gallery with Aspect Ratios** - Image grid with different ratios
15. **Responsive Order Change** - Mobile-first content reordering
16. **Group-Hover Reveal** - Hover-triggered content reveal
17. **Peer-State Toggle (Pure CSS)** - Checkbox-controlled button
18. **Dark Mode Toggle Simulation** - Dark mode styling example
19. **Form Validation UI** - Real-time input validation
20. **Full Dashboard Shell** - Complete admin dashboard layout

##  Getting Started

Simply open `index.html` in any modern browser. No build step required!

```bash
# Or use a local server
python -m http.server 8000
# Then visit http://localhost:8000
```

## 📦 Tech Stack

- **Tailwind CSS** (via CDN)
- **HTML5**
- **No JavaScript dependencies**

## 📝 Notes

- Uses Tailwind CSS CDN for rapid prototyping
- All exercises are self-contained in a single HTML file
- Responsive design demonstrated throughout
- Pure CSS interactions (no JavaScript required)
